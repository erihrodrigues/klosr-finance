## English

A web app for financial management and daily cash closing, built with love for my dad.

### 📱 About

**Klosr Finance** was born from a real need: help my dad manage the finances of his stores in a simple and organized way, right from his phone.

The app allows registering daily cash closings, tracking income by PIX, card and cash, monitoring expenses by category and generating PDF reports — all with a beautiful and intuitive interface.

### ✨ Features

- 🔐 Firebase authentication (login and register)
- 👥 Role system: Superadmin, Admin, Moderator and Employee
- 🏪 Multiple store management
- 💰 Daily cash closing (PIX, card, cash and expenses)
- 📊 Dashboard with financial summary
- 📦 Store inventory with custom categories
- 📄 PDF report export with monthly summary and expenses by category
- 🔔 Daily closing reminder via browser notification

### 🛠️ Tech Stack

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Firebase](https://firebase.google.com/) (Auth + Firestore)
- [Tailwind CSS](https://tailwindcss.com/)
- [jsPDF](https://github.com/parallax/jsPDF) + [jspdf-autotable](https://github.com/simonbengtsson/jsPDF-AutoTable)
- [React Router](https://reactrouter.com/)

### 🚀 Running locally

```bash
git clone https://github.com/erihrodrigues/klosr-finance.git
cd klosr-finance
npm install
npm run dev
```

> Create a `.env` file with your Firebase credentials before running.

## Português

Aplicativo web de gestão financeira e fechamento de caixa, desenvolvido com amor para o meu pai.

### 📱 Sobre o projeto

O **Klosr Finance** nasceu de uma necessidade real: ajudar meu pai a controlar o financeiro das lojas dele de forma simples e organizada, diretamente pelo celular.

O app permite registrar fechamentos diários de caixa, controlar entradas por PIX, cartão e dinheiro, acompanhar gastos por categoria e gerar relatórios em PDF — tudo com uma interface bonita e intuitiva.

### ✨ Funcionalidades

- 🔐 Autenticação com Firebase (login e cadastro)
- 👥 Sistema de cargos: Superadmin, Admin, Moderador e Funcionário
- 🏪 Gerenciamento de múltiplas lojas
- 💰 Fechamento de caixa diário (PIX, cartão, dinheiro e gastos)
- 📊 Dashboard com resumo financeiro
- 📦 Inventário por loja com categorias personalizadas
- 📄 Exportação de relatório em PDF com resumo mensal e gastos por categoria
- 🔔 Lembrete diário de fechamento via notificação do navegador

### 🛠️ Tecnologias

- [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Firebase](https://firebase.google.com/) (Auth + Firestore)
- [Tailwind CSS](https://tailwindcss.com/)
- [jsPDF](https://github.com/parallax/jsPDF) + [jspdf-autotable](https://github.com/simonbengtsson/jsPDF-AutoTable)
- [React Router](https://reactrouter.com/)

### 🚀 Como rodar localmente

```bash

cd klosr-finance

```
